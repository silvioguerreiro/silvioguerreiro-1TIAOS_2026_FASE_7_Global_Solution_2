# assets

Arquivos não estruturados do projeto: imagens, logotipos e figuras de resultado.

- `logo-fiap.png` — logotipo (placeholder; substituir pelo oficial).
- `figuras/` — saídas reais do MVP usadas no PDF e no README:
  - `fig_arquitetura.png` — arquitetura em camadas da solução.
  - `fig_cena_deteccoes.png` — detecção de mineração/queimada na cena orbital.
  - `fig_previsao.png` — previsão de focos de calor (RNN/LSTM).
  - `fig_rota_ga.png` — rota de patrulha otimizada por Algoritmo Genético.

As figuras são regeneradas por `src/make_figures.py`.
